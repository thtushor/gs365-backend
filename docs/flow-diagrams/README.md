## ğŸ”‘ User Roles

| Role | Description | Access Level |
|------|-------------|--------------|
| **Player** | End users who place bets | Play games, deposit, withdraw |
| **Affiliate** | Marketing partners | Refer players, earn commission |
| **Super Affiliate** | Senior affiliates | Manage affiliates, higher commission |
| **Agent** | Sales representatives | Manage players in regions |
| **Super Agent** | Senior agents | Manage agents |
| **Admin** | Platform administrators | Full platform management |
| **Super Admin** | System owners | Complete system access |
